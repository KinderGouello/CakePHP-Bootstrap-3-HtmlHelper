CakePHP-Bootstrap-3-HtmlHelper
==============================

* New features of BsHelper : [Nouveautés du BsHelper](http://webandcow.com/Page/Ressources/37/Bootstrap-et-CakePHP-10 "Nouveautés du BsHelper")
